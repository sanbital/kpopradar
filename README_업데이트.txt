K-POP PD 의사결정 레이더 v2 — 업데이트 안내

[이번 업데이트 핵심]
1. 국내곡 우선
- 기본값은 ‘국내곡 우선’입니다.
- 한국 지역·한국어 검색 결과와 국내 방송사·기획사·음원 유통 채널을 우선 반영합니다.
- ‘국내곡만 강하게’를 선택하면 국내 근거 점수가 낮은 후보를 분석에서 제외합니다.

2. 실재하지 않는 아티스트·곡 방지
- 제목에서 곡명/아티스트를 무작정 잘라내지 않고, 인용부호·커버 표기·아티스트-곡 구조·공식 영상 구조가 있을 때만 후보로 잡습니다.
- 상위 아티스트와 곡은 YouTube에서 다시 검색해 공식·방송·유통 채널 또는 여러 채널의 근거가 있는지 검증합니다.
- 검증을 통과한 후보만 ‘여보세요?’ 제작 조합에 사용합니다.
- 각 후보에 YouTube, Spotify, Apple Music, 멜론, 지니, 통합검색 링크가 붙습니다. 음원 서비스 링크는 직접 검색·확인용입니다.

3. 과거 명곡·커버 수요 확장
- 최신 컴백곡과 별도로 ‘과거 커버곡 탐색 기간’을 설정할 수 있습니다.
- 기본 1년이며 6개월·2년·5년으로 변경할 수 있습니다.
- 리무진서비스, 복면가왕, 비긴어게인, 킬링보이스, 노래방 라이브, 2000·2010년대 명곡 커버 등 별도 검색 묶음을 사용합니다.
- 여러 채널·여러 아티스트가 반복 커버한 곡, 오래된 곡의 재가속 신호를 커버 후보 점수에 반영합니다.

4. 채널 범위 확대
- 기본값은 ‘광역 — 팬·커버·예능 포함’입니다.
- 음악방송·공식·기획사뿐 아니라 라이브/커버 채널, 아이돌 예능, 팬캠·팬튜브·크리에이터, 쇼츠·챌린지까지 함께 봅니다.

5. 근거 링크
- 영상 제목, 채널, 출연자 근거, 곡 근거 영상에 직접 링크가 붙습니다.
- 곡 후보에는 YouTube·Spotify·Apple Music·멜론·지니·Google 통합검색 링크가 붙습니다.
- CSV에도 원본 영상 URL과 음원 서비스 검색 링크가 포함됩니다.

6. 프로모션/광고 유입 의심 필터
- 조회수·조회 속도·구독자 대비 도달은 큰데 좋아요율·댓글률·총반응률이 같은 유형 영상의 중앙값보다 지나치게 낮으면 의심 표시합니다.
- 기본값은 추천·순위 계산에서 제외입니다.
- ‘포함하되 표시’로 바꾸면 표에는 남기되 의심 사유가 표시됩니다.
- 이 판정은 광고 집행 사실을 증명하는 것이 아니라, 유기적 반응으로 보기 어려운 통계적 패턴을 거르는 휴리스틱입니다.

[사용 방법]
1. config.js를 메모장으로 엽니다.
2. YOUR_YOUTUBE_API_KEY_HERE만 실제 YouTube Data API 키로 교체합니다.
3. index.html, config.js를 GitHub 저장소의 같은 폴더에 올립니다.
4. 페이지를 연 뒤 ‘PD 편성안 생성’을 누릅니다.

[추천 기본 설정]
- 국내곡 기준: 국내곡 우선
- 채널 범위: 광역 — 팬·커버·예능 포함
- 후보 수집: 조회수+최신 균형
- 과거 커버곡 탐색: 1년 또는 2년
- 실재 검증: 사용
- 프로모션 의심: 추천·순위에서 제외
- 역주행 원곡 추적: 사용

[중요한 한계]
- Spotify·Apple Music·멜론·지니는 자동 수집이 아니라 직접 검색 링크입니다. 해당 서비스에서 곡·아티스트·원곡 정보를 최종 확인하세요.
- YouTube 공개 API에는 공유 횟수가 없습니다. 공유·인용 반응은 별도 X 보조신호 CSV/JSON을 넣었을 때만 보조 반영됩니다.
- 장르·곡명·아티스트 판정은 메타데이터 기반이므로 최종 캐스팅·선곡 전 원본 영상과 음원 서비스에서 확인해야 합니다.
- 프로모션 의심 필터는 확정 광고 판정이 아닙니다.
