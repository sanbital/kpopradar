K-POP PD 의사결정 레이더 — API 키 넣는 법

1. config.js를 메모장으로 엽니다.
2. YOUR_YOUTUBE_API_KEY_HERE 부분만 실제 API 키로 바꿉니다.
3. 저장합니다.
4. index.html과 config.js를 GitHub 저장소의 같은 위치에 올립니다.

예시
변경 전: youtubeApiKey: 'YOUR_YOUTUBE_API_KEY_HERE'
변경 후: youtubeApiKey: 'AIzaSy......실제키......'

주의
- 따옴표(' ')는 지우지 않습니다.
- index.html과 config.js는 반드시 같은 폴더에 둡니다.
- Google Cloud에서 웹사이트 제한과 YouTube Data API v3 제한을 설정하세요.
